# Telegram-уведомления Hookah Menu

Telegram используется только для уведомления мастера о новом гостевом заказе. Если Telegram временно недоступен, заказ всё равно сохраняется в Supabase и появляется во вкладке `Заявки гостей`.

## 1. Создайте бота

1. Откройте в Telegram официального бота `@BotFather`.
2. Отправьте команду `/newbot`.
3. Укажите отображаемое имя и уникальное имя, оканчивающееся на `bot`.
4. Сохраните полученный токен. Не отправляйте его в чат и не добавляйте в GitHub.

## 2. Получите идентификатор чата

1. Откройте созданного бота и нажмите `Запустить`.
2. Отправьте ему любое сообщение.
3. Получите `chat_id` безопасным способом через Telegram API или служебного бота для просмотра своего ID.

Для общего рабочего чата добавьте бота в группу, отправьте сообщение в группу и используйте идентификатор этой группы.

## 3. Добавьте переменные в Render

Откройте сервис Hookah Menu в Render, затем раздел `Environment` и добавьте:

```text
TELEGRAM_ENABLED=true
TELEGRAM_BOT_TOKEN=полученный_токен
TELEGRAM_CHAT_ID=идентификатор_чата
PUBLIC_APP_URL=https://hookah-menu-8cqq.onrender.com
```

Сохраните изменения. Render перезапустит сайт. Токен должен находиться только в Render и не должен иметь префикс `VITE_`.

## 4. Проверка

1. Войдите на сайт как гость.
2. Соберите и отправьте новый заказ.
3. Убедитесь, что заказ появился во вкладке мастера `Заявки гостей`.
4. Проверьте сообщение от Telegram-бота.

В Supabase у заказа поле `telegram_sent` станет `true`. При ошибке заказ сохранится, а краткое описание попадёт в `telegram_error`.
