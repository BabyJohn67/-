# Гостевые заказы: включение этапа 3

Код этапа 3 уже находится в проекте. Перед публикацией нужно один раз создать таблицу заказов в Supabase.

## 1. Создать таблицу

1. Откройте проект Hookah Menu в Supabase.
2. Перейдите в `SQL Editor`.
3. Откройте файл `supabase/migrations/002_guest_orders.sql` из проекта.
4. Скопируйте весь SQL из файла в редактор Supabase.
5. Нажмите `Run`.

Миграция создаёт таблицу `guest_orders`, индексы и правила доступа. Пароли и секретные ключи в SQL отсутствуют.

## 2. Проверить гостевой заказ

1. Откройте сайт как гость и войдите в обычный аккаунт.
2. Выберите формат, табаки и крепость.
3. Укажите имя, email и номер стола.
4. Нажмите `Отправить мой выбор`.
5. Проверьте заказ в окне подтверждения и нажмите `Подтвердить и отправить`.
6. В блоке `Мои заказы` появится номер и статус заказа.

## 3. Проверить панель мастера

1. Войдите в аккаунт администратора или мастера.
2. Откройте вкладку `Заявки гостей`.
3. Нажмите `Принять`.
4. Нажмите `Создать кальян`.
5. Выберите свободный физический кальян и сохраните микс.

Табак списывается только на последнем шаге, когда мастер сохраняет реальный активный микс. Простая гостевая заявка склад не изменяет.

## Статусы

- `Новый` — заявка только поступила.
- `Принят` — мастер подтвердил заказ.
- `Готовится` — кальян собирается.
- `Готов` — кальян готов к подаче.
- `Завершён` — работа с заказом закончена.
- `Отменён` — заявка отменена.

Telegram и email на этом этапе не подключаются.
